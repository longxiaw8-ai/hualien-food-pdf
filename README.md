# 🌸 花蓮美食 PDF 生成器

這是一個可以搜尋花蓮美食並產生 PDF 檔案的 Python 專案！

## 功能

- 🔍 自動整理花蓮美食清單
- 📄 產生美觀的 PDF 檔案
- 🖨️ 包含餐廳名稱、地址、類型、必點推薦

## 安裝

```bash
pip install fpdf requests beautifulsoup4
```

## 使用方式

```bash
python hualien_food.py
```

## 輸出

會在同目錄下產生 `花蓮美食推薦.pdf` 檔案。

## 範例

```
📍 液香扁食 - 花蓮市中山路136號
📍 炸彈蔥油餅 - 花蓮市重慶路&中山路交叉口
📍 公正街包子店 - 花蓮市公正街14-1號
...
```
